# 💱 Conversor Financeiro

Aplicação web moderna de conversão de moedas em tempo real, desenvolvida com HTML, CSS e JavaScript.

O projeto utiliza integração com API para buscar cotações atualizadas e oferece uma interface moderna inspirada em aplicações fintech.

---

## 🚀 Demonstração

🔗 Acesse o projeto:

[https://josemoraisrp.github.io/Conversor-de-moedas/](https://josemoraisrp.github.io/Conversor-de-moedas/)

---

## ✨ Funcionalidades

* ✅ Conversão de moedas em tempo real
* ✅ Integração com API de cotações
* ✅ Conversão dinâmica entre moedas
* ✅ Suporte para:

  * Real Brasileiro
  * Dólar Americano
  * Euro
  * Libra Esterlina
  * Iene Japonês
  * Bitcoin
* ✅ Histórico de conversões
* ✅ Botão de troca de moedas
* ✅ Atualização automática da cotação
* ✅ Interface responsiva
* ✅ Visual moderno com Glassmorphism
* ✅ Fundo animado em vídeo
* ✅ Microinterações e animações
